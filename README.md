# Taskflow Kanban

A React + Vite kanban board for tracking sprint work across backlog, active, review, and done lanes.

## Features

- Create tasks with assignee, priority, sprint, and summary
- Search tasks across title, summary, and assignee
- Filter board by priority
- Move tasks with drag and drop or quick-action buttons
- Persist board state in local storage

## Run locally

```bash
npm install
npm run dev
```

## Stack

- React 19
- TypeScript
- Vite
